# Ubuntu Youth Skills Hub Website

## Project Overview

Ubuntu Youth Skills Hub is a non-profit organisation website developed to provide information about youth skills development programmes, mentorship opportunities, and community initiatives. The website aims to empower unemployed youth by promoting digital skills training and career development opportunities.

## Features

* Home Page
* About Us Page
* Programmes Page
* Contact Us Page
* Donate Page
* Enquiry Form
* Search Functionality
* FAQ Accordion
* Read More Feature
* Contact Form Validation
* Enquiry Form Validation
* Donation Calculator
* Responsive Navigation Links
* External CSS Styling
* External JavaScript Functionality
* SEO Meta Tags
